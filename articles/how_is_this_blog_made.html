How is this blog made

12 April 2021 - Edited 17 July 2021

This blog is written in Org, on the magnificent VIM text editor.

It is compiled by a Ruby script, which is a pseudo SSG that is just translating Org to HTML with a link to a 10-lines-long CSS file.

Obviously, this site does not use any Javascript code and is available freely (as in freedom) at git.gnous.eu/wafelack/website.
